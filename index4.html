HTML Form Elements

SN Form Elements Description
1 Text Box Input Text box input is used to capture text such as a name, email, address or any other type of text
<input type="text" name="name" placeholder="Enter your name">
<input type="email" name="email" placeholder="Enter your email">
2 Password Input Password input does not show the text as the user types
<input type="password" name="password" placeholder="Enter your password">
3 Select Drop Down Drop Down specifies list of options that a user can choose when selecting a Drop Down menu
<select name="Programming Language">
 <option value="1">Python</option>
 <option value="2">C/C++/C#</option>
 <option value="3">Java/JavaScript</option>
</select>
Select your programming language
4 Color Input Color Input lets us pick a color
Select color
<input type="color" id="favcolor" name="favcolor" value="#000">
Select color     Selected
